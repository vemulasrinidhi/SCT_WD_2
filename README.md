# 🔢 Calculator Web Application

## 📌 Project Description

This project is a **modern and interactive Calculator Web Application** built using **HTML, CSS, and JavaScript**.
It performs basic arithmetic operations with a **beautiful glowing UI and smooth animations**.

---

## 🚀 Features

* ➕ Addition, ➖ Subtraction, ✖ Multiplication, ➗ Division
* 🎨 Attractive UI with **neon glow effects**
* ✨ Animated calculator title
* ⌨ Keyboard input support
* ❌ Error handling for invalid expressions
* 📱 Responsive design

---

## 🛠 Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & animations
* **JavaScript** – Logic & functionality
